## Common User Questions

### Q: What if I import a page that already has strokes but no transcription?
**A**: Transcribe it, then save. The dialog will show "âœ¨ New transcription" (purple badge).

### Q: What if I transcribe a page, notice an error, and fix it before saving?
**A**: Click Edit, fix the text, save the edit. When you save to LogSeq, it will still show "New transcription" since it's the first time saving text for that page.

### Q: Can I edit transcriptions that are already in LogSeq?
**A**: Yes! Import the page from LogSeq, click Edit on the transcription, make your changes, then save back to LogSeq. The dialog will show "ğŸ“ Updated transcription" (amber badge).

### Q: What happens if I save transcription changes without any stroke changes?
**A**: That works perfectly now! The dialog will only show the transcription change, and LogSeq will be updated with the new text while strokes remain unchanged.

### Q: Does editing transcription affect the original strokes?
**A**: No! Strokes and transcription are stored separately. Editing transcription text is purely a text operation and doesn't modify the underlying stroke data.
